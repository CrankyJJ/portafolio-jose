# Portafolio Personal - José Juan Dos Reis

## Desarrollador Web Fullstack | Graduado de DAW

Repositorio de mi portafolio personal: una web estática pensada para presentar mi perfil, habilidades y proyectos de forma clara, rápida y responsive.

---

## Características y stack

Proyecto **Front-End estático** enfocado en usabilidad, diseño responsive y rendimiento.

| Categoría | Tecnologías y herramientas |
| :--- | :--- |
| **Maquetación** | HTML5, CSS3 |
| **Estilos** | Tailwind CSS (CDN) + estilos personalizados |
| **Interactividad** | JavaScript (Vanilla) |
| **Control de Versiones** | Git / GitHub |
| **Iconos** | Devicon |
| **Diseño** | Estética dark con acentos neón |

---

## Estructura

Secciones principales (navegación por anclas/IDs):

1.  **Inicio / Hero (`#container`)**: presentación y CTA (ver proyectos / descargar CV).
2.  **Proyectos (`#projects`)**: cards con descripción, tecnologías y enlaces.
3.  **Habilidades (`#skills`)**: grid con stack.
4.  **Contacto (`#contact`)**: formulario y enlaces.

Estructura de carpetas (resumen):

```text
.
├─ index.html
└─ assets
   ├─ css
   ├─ img
   ├─ js
   └─ pdf
```

## Ejecutarlo en local

Si deseas clonar este repositorio para revisarlo o ejecutarlo en tu máquina local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/CrankyJJ/portafolio-jose.git
    ```
2.  **Acceder al directorio del proyecto:**
    ```bash
    cd portafolio-jose
    ```
3.  **Abrir el proyecto:**
    - Abre `index.html` directamente en el navegador, o
    - Usa un servidor local (recomendado) como la extensión **Live Server** de VS Code.
